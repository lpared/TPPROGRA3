Descripción

Este trabajo práctico consiste en el desarrollo de un sistema integrado para la administración y consulta de tarjetas de crédito. El proyecto está formado por una aplicación de consola desarrollada en C# y un portal web desarrollado en PHP. Ambas aplicaciones utilizan una misma base de datos MySQL para compartir la información de clientes, tarjetas y liquidaciones.

---

Tecnologías utilizadas

- C#
- .NET
- PHP
- MySQL
- HTML
- Tailwind CSS
- XAMPP

---

Funcionalidades implementadas

Aplicación de consola (Progra3)

- Alta de clientes.
- Emisión de nuevas tarjetas.
- Listado general de tarjetas.
- Consulta detallada de clientes y tarjetas.
- Eliminación de tarjetas.
- Generación de liquidaciones mensuales.

Portal Web (WebPHP)

- Activación de usuario web.
- Inicio de sesión.
- Consulta de la última liquidación.
- Consulta del historial de liquidaciones.

---

Base de datos

La base de datos utilizada es MySQL. El script para crear las tablas y cargar los datos iniciales se encuentra en el archivo mi_banco_db.sql.
